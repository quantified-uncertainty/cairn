## Get Started

This is a starter template for building research wikis with [Astro Starlight](https://starlight.astro.build/) and [Claude Code](https://docs.anthropic.com/en/docs/claude-code/overview).

1. Start the dev server: `npm run dev`
2. Open Claude Code: `claude`
3. Tell Claude your research topic and ask it to create your page structure

See the [Getting Started](/example/getting-started/) page for a detailed example prompt.
