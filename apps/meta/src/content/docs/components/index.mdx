## Available Packages

### @cairn/ui

Shared React components for CAIRN sites.

#### shadcn/ui Components

Import from `@cairn/ui/components`:

```tsx
import { Button, Card, Tabs } from '@cairn/ui/components';
```

Available components:
- Accordion
- Badge
- Button
- Card
- Collapsible
- DataTable
- HoverCard
- Input
- Popover
- Select
- Separator
- Table
- Tabs
- Tooltip

#### Graph Components

Import from `@cairn/ui/graph`:

```tsx
import CauseEffectGraph from '@cairn/ui/graph';
```

The CauseEffectGraph component provides:
- Interactive node-based visualizations
- Multiple layout algorithms (Dagre, ELK)
- Multiple view modes (Graph, List, Data, Mermaid)

### @cairn/tailwind

Shared Tailwind CSS theme tokens.

```css
@import "@cairn/tailwind/theme.css";
@import "tailwindcss";
```

Provides consistent colors, typography, and spacing across sites.
