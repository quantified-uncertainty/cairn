## What is LongtermWiki?

LongtermWiki is a framework for building structured knowledge bases with:

- **Astro + Starlight** for fast, static documentation sites
- **YAML-driven data** for structured content
- **React components** for interactive visualizations
- **LLM workflows** for content generation and improvement

## This Site

This meta-documentation explains how LongtermWiki works and how to build similar projects.
