Key projects in the epistemic tools and forecasting space include:

- **<EntityLink id="responses/epistemic-tools/projects/squiggle">Squiggle</EntityLink>**: Probabilistic programming language for uncertainty quantification
- **<EntityLink id="responses/epistemic-tools/projects/metaforecast">Metaforecast</EntityLink>**: Cross-platform forecast aggregation
- **<EntityLink id="responses/epistemic-tools/projects/squiggleai">SquiggleAI</EntityLink>**: LLM-powered probabilistic model generation
- **<EntityLink id="responses/epistemic-tools/projects/xpt">XPT</EntityLink>**: Structured adversarial collaboration on existential risks
- **<EntityLink id="responses/epistemic-tools/projects/forecastbench">ForecastBench</EntityLink>**: Dynamic benchmark for AI forecasting capabilities
- **<EntityLink id="responses/epistemic-tools/projects/ai-forecasting-benchmark">AI Forecasting Benchmark Tournament</EntityLink>**: Human vs AI forecasting competition

These projects aim to improve decision-making on high-stakes questions, particularly around AI development and existential risk.
